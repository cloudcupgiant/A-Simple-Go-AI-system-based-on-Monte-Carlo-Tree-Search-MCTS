# A-Simple-Go-AI-system-based-on-Monte-Carlo-Tree-Search-MCTS-.
This project implements a simple Go AI system based on Monte Carlo Tree Search (MCTS). Built on top of the provided Go rule framework, the project supports core game logic such as legal move checking, stone capturing, self-atari/invalid move detection, game termination, and scoring. On this foundation, it implements a random-move AI for basic rule verification and an MCTS-based AI for decision making on a 5×5 Go board. The MCTS agent follows the standard search procedure, including selection, expansion, simulation, and backpropagation, and further improves search efficiency through strategies such as heuristic move selection and limited simulation depth. The project also supports command-line matches between different agents and includes the design of a graphical interface for human-AI gameplay, allowing users to view the board, place stones, and interact with the AI. Overall, this project provides a complete course-oriented framework for implementing, testing, and analyzing basic Go AI algorithms.
